这是一道 K-Dtree 的模板题，题目在这里.

首先，这道题目强制在线，而且空间只有 20MB, 因此需要一个优秀的在线算法，甚至这个矩阵都不能渲染。

将一个二元组 $(x,y)$ 作为结点放入 K-Dtree, 代表某一个值不为 0 的点的坐标，同时维护一些额外的信息：

  • val 这个点目前的值
  • size 子树大小
  • x_1, y_1, x_2, y_2 所代表矩形的左下和右上点的坐标
  • sum 代表矩形中所有数的和

这样，我们在 K-Dtree 中可以实现插入和修改，同时像线段树中那样查询，即每到达一个结点，判断这个结点代表的矩形是否完全在目标矩形中，如果是则累加答案，如果完全不包含返回 0，如果交叉则左右递归。
